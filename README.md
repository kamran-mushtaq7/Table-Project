# 📊 HTML Tables Project

A simple and responsive web project demonstrating the creation and styling of HTML tables using semantic HTML and inline CSS. This project focuses on structuring tabular data effectively for better readability and presentation.

---

## 📌 Overview

This project showcases how to design and organize data using HTML `<table>` elements, along with basic inline CSS styling for layout and visual enhancement. It is ideal for beginners learning how to present structured data on web pages.

---

## 🚀 Features

* Well-structured HTML tables
* Use of table elements (`<table>`, `<tr>`, `<th>`, `<td>`)
* Inline CSS styling for borders, spacing, and alignment
* Clean and readable layout
* Beginner-friendly implementation

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3 (Inline Styling)**
